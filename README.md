# Pulse — Product Analytics Dashboard

A demo B2B SaaS dashboard built with Next.js (App Router), Tailwind CSS v4, and Recharts.
Design system generated with the [`ui-ux-pro-max`](.claude/skills/ui-ux-pro-max) skill (Real-Time
Operations pattern, dense dashboard layout, Fira Sans / Fira Code typography, light/dark tokens).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## What's here

- Responsive sidebar + topbar shell with a mobile slide-over nav
- Light/dark theme toggle (`next-themes`), no dark-mode-by-default
- KPI stat cards with trend indicators
- Trend area chart and horizontal comparison bar chart (Recharts), each with an
  accessible summary and a table fallback
- Sortable activity table with severity/status badges

## Stack

Next.js 16, React 19, TypeScript, Tailwind CSS v4, Recharts, lucide-react, next-themes.
